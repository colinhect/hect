enum hect::ComponentEventType

A component-related event type.

Enumerator
ComponentEventType_Add 

A component was added to an entity.

This event type is triggered when an entity pending activation is activated during Scene::refresh(). It also may be triggered immediately if an activated entity has a component added or replaced.

ComponentEventType_Remove 

A component was removed from an entity.

This event type is triggered when an entity pending destruction is destroyed during Scene::refresh(). It also may be triggered immediately if an activated entity has a component removed or replaced.
