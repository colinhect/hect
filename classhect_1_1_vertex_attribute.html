Detailed Description

An attribute of a vertex (position, normal, etc).

Note
A vertex attribute is made up of a semantic, a type, and a cardinality. The semantic describes how the attribute is used. The type describes the type of each component in the attribute. The cardinality is number of components (of the attribute's type) the attribute has.
