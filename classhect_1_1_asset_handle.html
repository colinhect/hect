Detailed Description

template<typename T>
class hect::AssetHandle< T >

A reference-counted handle for an Asset.

The prefered alias for this type is Asset::Handle.

Note
An asset handle may reference an asset which was loaded from the file system and is cached in an asset cache. Alternatively, it may reference a manually created asset. A handle works much like a smart pointer.
